
# Pages in Persepolis

Pages are written in Markdown.

For example, here's what an example page looks like (`collab/page.mdx`)

```md

# Collaborate with me!

## If you're interested in working together, let me know!

For example, my [design](/design) skills are pretty weak. I *LOVE* good design, but just don't spend so much time on it. I'd love to work with you on something cool.


```
